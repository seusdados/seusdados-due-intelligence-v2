## Introdução

O Sistema de Avaliações Unificadas permite realizar avaliações de conformidade, maturidade e risco de forma integrada, utilizando múltiplos frameworks normativos como LGPD, ISO 27001 e NIST CSF.

### Principais Funcionalidades

- ✅ Criação de avaliações com ID único
- ✅ Atribuição de domínios a respondentes
- ✅ Upload de evidências (PDF ou link)
- ✅ Análise de risco multi-norma
- ✅ Gráficos radar de maturidade
- ✅ Sistema de notificações automáticas
- ✅ Liberação de resultados para sponsors

## Primeiros Passos

### Acessando o Sistema

1. Acesse a plataforma através do link fornecido
2. Faça login com suas credenciais
3. Você será direcionado ao dashboard principal

### Navegação

O menu lateral contém as seguintes opções:

- **Dashboard**: Visão geral das avaliações
- **Avaliações**: Lista de todas as avaliações
- **Minha Conta**: Configurações do perfil

## Para Consultores

### Criando uma Nova Avaliação

1. Clique em **"Nova Avaliação"**
2. Selecione o **Framework** desejado:
   - Seusdados (padrão)
   - Conformidade LGPD
   - Misto
   - SGD
   - ICO
   - CNIL
3. Defina o **Prazo** (padrão: 15 dias)
4. Clique em **"Criar"**

> 💡 **Dica**: O ID da avaliação é gerado automaticamente no formato AC#YYYYMMDDHHMMXXX

### Atribuindo Domínios

1. Acesse a avaliação criada
2. Clique em **"Atribuir Domínios"**
3. Selecione os respondentes para cada domínio
4. Use **"Selecionar Todos"** para atribuir todos os domínios a um respondente
5. Clique em **"Confirmar Atribuições"**

### Revisando Respostas

1. Acesse o **Painel do Consultor**
2. Na aba **"Respostas"**, visualize todas as respostas recebidas
3. Verifique as evidências anexadas
4. Identifique pendências

### Editando Análise de Risco

1. Acesse a aba **"Análise de Risco"**
2. Selecione o nível de risco para cada domínio
3. Adicione referências normativas (LGPD, ISO, NIST)
4. Defina probabilidade e impacto
5. Clique em **"Salvar Análise"**

### Liberando Resultados

1. Acesse a aba **"Liberar Resultado"**
2. Selecione os destinatários:
   - ✅ Sponsor (obrigatório)
   - ☐ Membros do CPPD (opcional)
3. Configure as notificações:
   - ✅ Email com link
   - ✅ Anexar PDF
   - ☐ Notificação in-app
4. Clique em **"Liberar Resultado"**

## Para Respondentes

### Recebendo Notificações

Você receberá notificações por email quando:

- For atribuído a novos domínios
- Faltarem 10, 5, 2, 1 dia(s) para o prazo
- O prazo vencer hoje

### Respondendo o Questionário

1. Acesse o link recebido por email
2. Selecione o domínio a responder
3. Para cada questão:
   - Selecione o nível de maturidade (0-5)
   - Adicione observações (opcional)
   - Anexe evidências (se necessário)
4. Clique em **"Próxima"** para avançar
5. Ao final, clique em **"Finalizar Domínio"**

### Anexando Evidências

1. Clique em **"Adicionar Evidência"**
2. Selecione o tipo:
   - **PDF**: Arraste o arquivo ou clique para selecionar (máx. 10MB)
   - **Link**: Cole a URL do documento
3. Clique em **"Confirmar"**

> ⚠️ **Atenção**: Você não poderá finalizar sem anexar todas as evidências obrigatórias

### Visualizando Pendências

Se houver evidências pendentes:

1. Um aviso será exibido na tela
2. Clique em **"Ver Pendências"**
3. A lista mostrará todas as questões com evidências faltantes
4. Clique em **"Adicionar Evidência"** para cada item

## Para Sponsors

### Visualizando Resultados

Após a liberação pelo consultor:

1. Você receberá um email com o link
2. Acesse a plataforma
3. Vá para **"Resultados"**
4. Selecione a avaliação

### Entendendo o Dashboard

O dashboard de resultados inclui:

- **Gráfico Radar**: Maturidade por domínio
- **Análise de Risco**: Matriz com severidades
- **Plano de Ação**: Ações recomendadas com prazos
- **Resumo Executivo**: Visão geral da avaliação

### Exportando Relatórios

1. Clique em **"Exportar"**
2. Selecione o formato:
   - **PDF**: Relatório completo
   - **PNG**: Apenas gráficos
3. O download iniciará automaticamente

## Perguntas Frequentes

### Como alterar o prazo de uma avaliação?

Apenas consultores podem alterar prazos. Acesse a avaliação, clique em **"Editar"** e modifique a data de prazo.

### Posso redistribuir um domínio para outro respondente?

Sim. O consultor pode acessar **"Atribuições"** e reatribuir domínios a qualquer momento antes da finalização.

### O que acontece se eu não responder no prazo?

Você continuará recebendo notificações. O consultor será informado sobre o atraso e poderá entrar em contato.

### Como sei quais evidências são obrigatórias?

Questões com evidências obrigatórias são marcadas com um ícone de 📎. Você verá um aviso ao tentar finalizar sem anexá-las.

### Posso editar uma resposta já enviada?

Sim, desde que o domínio não tenha sido finalizado. Acesse a questão e clique em **"Editar"**.

### Como faço para solicitar suporte?

Entre em contato através do email suporte@seusdados.com.br ou utilize o chat de suporte na plataforma.

## Glossário

| Termo | Definição |
|-------|-----------|
| **Avaliação** | Processo de análise de conformidade ou maturidade |
| **Domínio** | Área temática dentro de um framework |
| **Framework** | Conjunto de critérios e questões para avaliação |
| **Evidência** | Documento que comprova uma resposta |
| **Maturidade** | Nível de desenvolvimento em uma área (0-5) |
| **Sponsor** | Responsável principal pela organização avaliada |
| **CPPD** | Comitê de Proteção de Dados Pessoais |
